FCC Image Search Abstraction Layer

Usage

  • Use https://yukixoma-fcc-image-searcher.herokuapp.com/api/imagesearch/[Search string] to search
    Example: https://yukixoma-fcc-image-searcher.herokuapp.com/api/imagesearch/anime

  • Use ?offset=[page] to paginate search result
    Example: https://yukixoma-fcc-image-searcher.herokuapp.com/api/imagesearch/[Search string]?offset=3

  • Use https://yukixoma-fcc-image-searcher.herokuapp.com/lastest to get lastest search history

Return value is in an JSON object
For easy to read use JSON reader (extension or online)
